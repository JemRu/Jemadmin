# Jemadmin
##  山东大学数据库课程设计
基于ace组件的 查询信息、户口管理、人员管理 小型户籍管理系统

1. 户籍的查询
2. 人口的查询
3. 人口信息修改
4. 该人口的户口信息修改
5. 人口的迁入迁出信息修改
6. 人口的新建
7. 人口的注销
8. 户籍信息的修改，
9. 该户下个人信息的修改
10. 户籍的迁入迁出修改
11. 户籍的新建
12. 户籍的注销

* code: 代码包
* UI: 网站部分界面截图
* doc: 项目各阶段文档说明
* conf.txt: 配置说明
